# Vertical Slider
Project #26 in Brad Traversy's 50 Projects in 50 Days. This is a vertical slider. Clicking the arrows changes the images and content. This project is not responsive.

![Vertical Slider](screenshot.jpg)

## Built With
- [HTML5](https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/HTML5)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)
- [Javascript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)

## Install
```
    $ git clone git@github.com:samiiexx/sam-js-26.git
    $ cd main
```
